Here are some of my projects

  • Easy Work - a project tracker

    Html, CSS, JavaScript, React, Firebase

    This project was created mainly with React and it uses Firebase for authentication, storage, and hosting. The purpose of this webapp was to create a JIRA like environment. It is rather barebones compared to Jira but the functionality is similar. You can make an account, add/complete projects, view online team members and also comment on current tasks.

  • Weather App

    Html, CSS, JavaScript, React, NodeJs, ExpressJs, Heroku

    This is a weather app I made that kind of resembles the native iOS weather application. This uses a free weather api to get the data. The data is processed using NodeJs and Express Js on the backend. The frontend is done using React Hooks and it is hosted on Heroku.

  • Money Manager - Finance Tracker

    Html, CSS, JavaScript, React, Firebase

    This little webapp helps me keep track of my spending habits. I used React and Firebase for the authentication, storage, and deployment. This was my first attempt at using firebase for all three and it worked out really well. To use the app, you must sign up with an account to access you personalized spending tracker.
